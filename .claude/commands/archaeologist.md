# 🏺 Skill: Archaeologist | CoreBrain Reverse Engineering

Escavação profunda de módulos e análise de fluxos técnicos.

---

## 🎯 Objetivo
Analisar o código-fonte para extrair algoritmos, lógica de controle e estruturas de dados (Escavação).

## 🛠️ Processo (Inspirado em Reversa)
1.  **Análise de Fluxo**: Mapear entrada/saída de dados e chamadas de funções.
2.  **Lógica Complexa**: Desvendar regras de negócio enterradas em condicionais ou loops.
3.  **Contratos Reais**: Documentar como os módulos se comunicam na prática (mesmo sem tipos).
4.  **Diagramação**: Gerar Mermaid Sequence Diagrams para fluxos críticos.

## 📝 Output Esperado
Em `memory/specifications/`:
- `code-analysis.md`: Análise técnica por módulo com diagramas de sequência.

## 🟢 Traceability Marks
- 🟢 **CONFIRMED**: Extraído diretamente do código (citar arquivo:linha).
- 🟡 **INFERRED**: Lógica deduzida por contexto.
- 🔴 **GAP**: Código ofuscado ou comportamento incerto.

---
**Ativação:** `brain, use archaeologist em [módulo]`
