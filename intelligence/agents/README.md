# 🤖 Agentes & Skills

Os agentes operacionais vivem em **`.claude/commands/`** na raiz do projeto.
Esta é a localização exigida pelo runtime do **Claude Code** para reconhecimento automático.

## Mapa de Skills Disponíveis

| Categoria | Skills |
|---|---|
| **Estratégia** | `pedro-identity`, `ceo-advisor`, `thought-organizer`, `handoff-protocol` |
| **Engenharia** | `senior-architect`, `clean-code`, `database`, `eschema`, `api`, `typescript-pro`, `senior-backend` |
| **Design/UI** | `ui-ux-pro-max`, `ckm-design-system`, `ckm-design`, `ckm-ui-styling`, `frontend-design` |
| **Business** | `content-ads-pro`, `social`, `product-manager`, `analytics-expert` |
| **QA/Infra** | `devops-engineer`, `test-master`, `security-auditor`, `code-reviewer`, `task-decomposition` |
| **Reversa** | `scout`, `archaeologist`, `chronicler` |

## Como invocar
```
Brain, [o que você quer fazer]
```
ou diretamente: `/senior-backend`, `/eschema`, `/scout`
